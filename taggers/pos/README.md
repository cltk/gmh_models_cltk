## File format

All tokens are normalized according to [Referenzkorpus Mittelhochdeutsch](https://linguistics.rub.de/forschung/arbeitsberichte/19.pdf)'s rules.

* norm_pos.pickle: [(token1, {pos1_1, pos1_2, ...}), (token2, {pos2_1, pos2_2, ...}), ...]
